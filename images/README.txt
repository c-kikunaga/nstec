【画像ファイル配置ガイド】

■ ヒーロー（最重要）
  hero-main.jpg   … チャットで添付いただいた工場作業の写真（1枚目）
                    推奨サイズ：1280x640px 以上
                    配置後、index.html の <img src="images/hero-main.jpg"> がそのまま反映されます

■ 仕事内容
  job-assembly.jpg   … 組立作業の様子
  job-press.jpg      … プレス作業の様子
  job-inspection.jpg … 検査作業の様子
  配置後、index.html 内の該当コメントアウトを解除してください

■ 推奨フォーマット
  JPEG / WebP（品質70〜80%）
  カード画像：600x400px

■ 画像がない場合
  絵文字プレースホルダーが自動表示されます（本番前に差し替えを推奨）
